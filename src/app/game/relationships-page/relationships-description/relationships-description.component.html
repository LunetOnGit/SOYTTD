This is where you can tell us about the trust levels between characters.
By setting these levels, you can ensure that characters who dislike each other won't cooperate, and those who trust each other will work together.

I recommend setting a default trust level that seems reasonable for most characters and adjusting only where necessary.
For instance, the current default is 50% (neutral) because most characters don't know each other. I've only changed the trust levels for characters who already know each other.

Here's a guide to the trust levels:
0-20%: Strong Distrust
21-40%: Distrust
41-59%: Neutral
60-80%: Trust
81-100%: Strong Trust
